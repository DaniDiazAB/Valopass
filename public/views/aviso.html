Aviso Legal — Live Demo de ValoPass

Advertencia sobre el uso del entorno de demostración (live demo) y el tratamiento de datos.

IMPORTANTE — DEMO NO SEGURO

Esta versión en línea es únicamente una demo para mostrar el funcionamiento de la aplicación. Aunque no se almacenen datos se recomienda no introducir ningún tipo de información

¿Cómo se almacenan los datos?

En esta demo, las cuentas de Valorant (las que aparecen en la lista de "cuentas") no almacenan las contraseñas cifradas en la base de datos. Es decir, las contraseñas relacionadas con las cuentas que se ven en la demo están guardadas en texto plano en la base de datos de demostración. Por el contrario, las contraseñas de los usuarios que se registran en la aplicación (login) sí están encriptadas según el comportamiento actual de la aplicación.

Resumen rápido

  • No almacenar datos sensibles:, aunque no se guarden evita introducir contraseñas reales en la demo.
  • Demo visible públicamente: la demo está pensada para mostrar funcionalidad, no para operar en producción.
  • Código abierto: puedes descargar, revisar y modificar el código desde el repositorio del proyecto.
Recomendaciones de seguridad (para quien use o instale el código):
  • Usa password_hash / password_verify en PHP para almacenar contraseñas de forma segura.
  • Aplica consultas preparadas / parámetros enlazados para evitar inyecciones SQL.
  • Protege la aplicación con HTTPS y controla el acceso a la base de datos.
  • Si publicas una instancia en producción, encripta todas las contraseñas y datos sensibles, y no reutilices contraseñas reales de usuarios.
